\section{Weeding}
\begin{definition}
Weeding checks language rules that could be checked by CFG, but easier to check with code
\end{definition}

\section{Intermediate Representations}

\begin{definition}
\textbf{Intermediate representation} (IR) is the data structure representing program being compiled
\end{definition}

\begin{itemize}
\item A parse tree has many redundant nodes with non-terminals
\item A abstract tree is a structure which optimizes the parse tree and stores types and values
\item Build AST using recursive traversal of parse trees
\item Other types of IRs
\begin{itemize}
\item Instruction-based IRs
\item Graph-based IRs
\end{itemize}
\end{itemize}

\section{Context-sensitive analysis}

\subsection{Name Resolution}
\begin{itemize}
\item The goal is to connect every use of a name to the true meaning 
\item To determine which instance of a name is associated with a reference
\item Class methods and inherited methods need to be caught
\end{itemize}

\begin{definition}
An \textbf{environment} is a map from names to their meanings
\end{definition}

\begin{definition}
A \textbf{Scope} is an area of the program where a declaration has effect (is visible)
\begin{itemize}
\item Associated with AST node
\item each scope has an environment
\item pointer to outer (enclosing scope)
\end{itemize} 
\end{definition}

\subsubsection{Name Lookup (use)}
\begin{itemize}
\item Search in current environment
\item not found, recursively search enclosing scopes
\item if not found throw error
\end{itemize}

\subsubsection{Declaring a name}
\begin{itemize}
\item if name in current environment, throw error
\item Insert into current environment 
\end{itemize}

\subsubsection{Name spaces}
\begin{itemize}
\item There is usually multiple environments for some scope
\item The environment for decoding/lookups is determined syntactically
\item Namespaces in JAVA
\begin{itemize}
\item Package Name Space
\item Type (Class or Interface)
\item Method Name Space
\item Expression Name Space (Variable, parameters, fields)
\end{itemize}
\end{itemize}
