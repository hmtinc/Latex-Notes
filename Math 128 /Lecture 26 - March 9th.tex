\section{Comparison Test}
\begin{theorem}
Suppose that \(\sum a_n\) and \(\sum b_n\) are series with positive terms and \(a_n \leq b_n\) doe all all n.
\begin{enumerate}
\item If \(\sum b_n\) converges, then \(\sum a_n\) converges
\item If \(\sum b_n\) diverges, then \(\sum b_n\) diverges 
\end{enumerate}
\end{theorem}

\textbf{Notes : } In order to use this test, we have to first guess if the series converges or diverges

\section{The Limit Comparison Test (LCT)}
\begin{theorem}
consider \(\sum a_n\) and \(\sum b_n\). Suppose \(a_n > 0\) and \(b_n > 0\) for all n.\\
If \(\lim_{n\to \infty} \frac{a_n}{b_n} = L\), where L is a finite number and \(L > 0\), then either both series converges or both diverge
\end{theorem}
