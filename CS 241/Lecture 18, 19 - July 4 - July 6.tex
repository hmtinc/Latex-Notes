\section{Code Generation}
The purposes of code generation is to actually generate machine code based on the given code 

Key issues include 
\begin{itemize}
\item Correctness
\item Efficiency
\item Efficiency of the compiled code
\end{itemize}

To solve these key issues we approach this using \textbf{Syntax Direct Translation}

\begin{itemize}
\item create a translation function for each syntactic category, e.g.
for loops, if-else statements, assignment, expressions
\item translation closely follows the syntactic structure of the code
\item recursively traverse the parse tree to gather the needed
information

\end{itemize}
