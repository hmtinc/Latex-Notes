\begin{definition}
A \textbf{Closed Walk} is one which \(v_0 = v_n\). \\
A \textbf{Cycle} is a walk where all \(v_0, v_1,\ldots, v_{n-1}\)
are distinct and \(v_0 = v_n\)
\end{definition}

\begin{theorem} 
Let \(x, y \in V(G)\). If there is a walk from x to y, then there is a path from x to y
\end{theorem}

\begin{proof}
Consider a walk \(W: v_nc_1, v_1 c_2, \ldots, v_{n-1} c_{n}\) where \(v_0 = x\), \(v_n = u\) and suppose length(W) is as small as possible.
\begin{itemize}
\item If W is a path, we are done.
\item Suppose W is not a path, then there exists \(0 \leq i \leq j \leq n\) such that \(v_i = v_j\). Then, \(W^\prime = v_0v_1\ldots, v_i v_{j+1} v_{j+2} \ldots v_n\) has a shorter length than W, which is a contradiction. 
\end{itemize}
\end{proof}

\begin{theorem}
If every vertex in G has degree at least 2, then G has a cycle. 
\end{theorem}

\begin{proof}
Take a path \(P = v_0v_1\ldots v_1\) of maximal length, \(v_0\) has a neighbour \(Z \neq v_1\). \\
Such Z must  be in P otherwise \(P^prime : zv_0v_1\ldots v_n\) would be larger than P. \\
Thus \(v_0\) has a neighbour \(v_i\) with \( i = 2, \ldots, n\). Then \(v_0v_1\ldots v_i v_0 \) is a a cycle
\end{proof}

\begin{itemize}
\item Hamilton Cycle : A cycle that goes through all the vertices of the graph 
\end{itemize}
