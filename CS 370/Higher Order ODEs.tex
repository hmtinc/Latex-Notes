\lecture{5}{Higher Order ODEs}{Christopher Batty}{Harsh Mistry}
\begin{itemize}
\item We often encounter ODE's with higher order derivatives (i.e oscillating springs, pendulums, etc)
\item the order of the ODE is the highest derivative that appears
\end{itemize}

\section{Converting to First Order}
For each variable \(y\) with more than a first derivative, introduce new variables
$$ y_i = y^{(i-1)}$$
for i = 1 to n, so each derivative has a corresponding new variable \\
substituting the new variables into the original ODE leads to 
\begin{enumerate}
\item One first order equation for each original equation
\item One or more additional equations relating the variables
\end{enumerate}
