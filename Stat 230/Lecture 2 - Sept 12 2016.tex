\section{What is Probability?}

\subsection*{The classical definition}

The chance of a specific event occurring where all events have an equal chance of occurring.
\begin{center}
\(\frac{\textbf{Number of ways the event can occur}}{\textbf{The total number of outcomes}}\)
\end{center}


\subsection*{The relative-frequency definition}

The chance of a specific outcome occurs in relation to the proportion of times it occurs over the long run.

\subsection*{The subjective probability definition}

The degree to which a given individual believes the event will happen 

\begin{ablock}{Note}
Subjective  Probability, since it is personal, there is no â€œsingle correct answer".
\end{ablock}

\subsection{Sample Spaces and Events}

\begin{itemize}
\item \textbf{Experiment}  is any action, phenomenon or process that can be infinitely repeated, at least in theory.
\item An experiment is said to be \textbf{random} if it has more than one possible outcome, and \textbf{deterministic} if it has only one.

\begin{itemize}
\item Tossing a coin once or several times
\item Obtaining blood types from a group of individuals
\end{itemize}

\item \textbf{Trial} is a single repetition of the experiment.

\item \textbf{Sample Space} of an experiment denoted by S, is the set of all possible \textbf{distinct} outcomes of that experiment.
\end{itemize}

\begin{ablock}{Notes}
\begin{itemize}
\item In a single trial, one and only one of the outcomes in S can occur.
\item Use {} to indicate the elements of a set.
\end{itemize}
\end{ablock}

\subsection{Sample Spaces}

As noted a sample space the set of all possible distinct outcomes In addition sample spaces can be continuous or discrete.
