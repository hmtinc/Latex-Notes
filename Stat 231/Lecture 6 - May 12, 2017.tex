\section{Correlation Coefficient}
The correlation coefficient measures the degree of linear association between X and Y.
\[\text{ Sample Correlation Coefficient } =  r_{xy} = \frac{\sum_{i = 1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{
\begin{bmatrix} \sum_{i = 1}^n (x_i - \bar{x})^2 \end{bmatrix}
\begin{bmatrix} \sum_{i=1}^n (y_i - \bar{y})^2 \end{bmatrix}}} \]

\subsection{Properties of the Correlation Coefficient}
\begin{itemize}
\item \(\mid r_{xy} \mid \hspace{0.1cm}\leq 1 \) 
\item If \(y_i = a + b x_i , \forall i = 1, \ldots, n\) 
\[ r_xy = \begin{cases} 1 \text{ if } b > 0 \\ -1 \text{ if } b < 0 \end{cases}\]
\item The sign of the value determines direction 
\item the value represents the strength of the linear relationship 
\item Higher values of \(\mid r \mid\) represent a stronger linear association 
\item \(\mid r \mid \hspace{0.1cm} \approx 0 \) indicates evidence of no linear association
\item Ever if \(\mid r \mid \) is really high, we cannot conclude \underline{Causation} without further analysis
\end{itemize}
